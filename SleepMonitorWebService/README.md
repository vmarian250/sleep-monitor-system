SleepMonitor System

-> Overview
Client-server application for monitoring and analyzing sleep data.

->Features
- CRUD operations for sleep records
- Data visualization (charts for sleep quality)
- Personalized recommendations based on history
- Client-server architecture

->Tech Stack
- Backend: ASP.NET Web Service
- Frontend: C# (Windows Forms)
- Database: SQL Server

->Highlights
- Designed relational database with optimized queries
- Implemented asynchronous client-server communication
- Built recommendation system based on historical data
- Developed graphical data visualization

->How to Run
1. Clone the repository
2. Setup SQL Server database
3. Run backend service
4. Start Windows Forms client
